## 📦 数据库现状

### 表结构

| 表名 | 说明 | 主要字段 |
|------|------|---------|
| `staff` | 教职工（管理员+教师） | id, username, password_hash, password_plain, name, **role**(teacher/admin), gender, subject |
| `students` | 学生 | id, **username**(学号), password_hash, password_plain, name, gender, **class_id** |
| `classes` | 班级 | id, name, **section**(小学部/初中部), grade, homeroom_teacher_id → staff.id |

### 数据量

| 类型 | 数量 | 说明 |
|------|:----:|------|
| 管理员 | 1 | admin / admin123 |
| 教师 | 12 | 小学部三年级3人+四年级3人，初中部一年级3人+二年级3人 |
| 班级 | 8 | 小学部三年级×2、四年级×2，初中部一年级×2、二年级×2 |
| 学生 | 240 | 每班30人（男女各15），学号 01030101 ~ 02020230 |
| 共计 | 253 用户 | |

### 学号格式

```
SS GG CC NN
学部 年级 班级 序号

01=小学部  02=初中部
01=一年级  02=二年级  03=三年级  04=四年级
01=一班    02=二班
01~30=班级内序号

示例:
  02010120 → 初中部一年级二班第20人
  01030105 → 小学部三年级一班第05人
```
