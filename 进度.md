## 📦 数据库现状

### 表结构

| 表名       | 说明                  | 主要字段                                                                                                                             |
| ---------- | --------------------- | ------------------------------------------------------------------------------------------------------------------------------------ |
| `staff`    | 教职工（管理员+教师） | id, username, password_hash, password_plain, name, **role**(teacher/admin), gender, **subject**(chinese/math/english/science/ethics) |
| `students` | 学生                  | id, **username**(学号), password_hash, password_plain, name, gender, **class_id**                                                    |
| `classes`  | 班级                  | id, name, **section**(小学部/初中部), grade, homeroom_teacher_id → staff.id                                                          |
| `exams`    | 考试                  | id, name, **class_id** → classes.id, exam_date, created_at                                                                           |
| `scores`   | 成绩                  | id, student_id → students.id, exam_id → exams.id, **subject**, **score**, class_rank, school_rank                                    |

### 数据量

| 类型   | 数量 | 说明                                                                 |
| ------ | :--: | -------------------------------------------------------------------- |
| 管理员 |  1   | admin / admin123                                                     |
| 教师   |  26  | 小学部三年级~六年级各 5 科 (20人) + 初中部一年级~二年级各 3 科 (6人) |
| 班级   |  12  | 小学部三年级~六年级×2（8个）+ 初中部一年级~二年级×2（4个）           |
| 学生   | 360  | 每班30人（男女各15），学号 01030101 ~ 01060230                       |
| 考试   |  6   | 一班：第一次月考~期末考试（2024-10 ~ 2025-01）                       |
| 成绩   | 2700 | 6次考试 × 5科 × 90人（一班+二班），decimal(5,2)                      |

### 学号格式

```
SS GG CC NN
学部 年级 班级 序号

01=小学部  02=初中部
01=一年级  02=二年级  03=三年级  04=四年级  05=五年级  06=六年级
01=一班    02=二班
01~30=班级内序号
```
