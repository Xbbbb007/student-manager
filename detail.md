# 智慧学生管理系统 — 功能详细说明

本文档详细介绍各角色登录后可使用的所有功能，以实际菜单结构为准。

---

## 一、通用功能（三端共有）

### 用户登录

系统启动后进入登录界面，输入用户名与密码完成验证。

- 密码以 **MD5 + 随机盐值** 加密存储，系统不保存明文
- **连续登录失败 5 次** 触发账号锁定，防止暴力破解
- 登录成功后根据角色自动跳转对应主界面（管理员 / 教师 / 学生）
- 任何界面均可通过选择 `[0] 返回` 逐级退出，直至退出系统

管理员拥有系统最高权限，主菜单共 9 个模块。

```

  [5] 教学计划管理
  [6] 课表排课管理
  [7] 公告通知管理
  [8] 账户与安全管理
  [9] 数据统计与报表
```

### 1. 学生管理

```
  [1] 查看学生花名册（分页 / 条件查询）
  [2] 添加新学生
  [3] 修改学生信息
  [4] 删除学生归档
  [5] 学生转班操作
  [6] CSV 批量导入学生
  [7] 导出学生到 CSV
```

| 功能 | 详细说明 |
|------|----------|
| **花名册查询** | 支持按学号/姓名/班级/专业模糊搜索，分页浏览（每页10条），表格对齐展示 |
| **添加学生** | 填写学号、姓名、性别、出生日期、手机号、邮箱、地址、入学日期，选择所属班级，自动创建对应登录账户 |
| **修改信息** | 按学号查找后修改可编辑字段（手机/邮箱/地址等），学号和班级通过专项操作变更 |
| **删除归档** | 软删除（status=0），数据保留便于追溯，需 y/n 二次确认 |
| **转班操作** | 输入学生学号和目标班级编号，完成班级迁移 |
| **CSV 导入** | 按系统规定格式的 CSV 文件批量创建学生记录，自动跳过重复学号 |
| **CSV 导出** | 将当前查询结果或全部学生信息导出为 CSV 文件 |

```
  [1] 查看教学计划列表
  [2] 创建新教学计划（开课安排）
  [3] 修改教学计划
  [4] 删除教学计划
```

| 功能 | 详细说明 |
|------|----------|
| **创建计划** | 选择课程、指定授课教师、选择授课班级（选修课可不绑定班级）、设置学期，设置最大选课人数 |
| **计划列表** | 显示学期、课程名、授课教师、授课班级、当前选课人数/最大容量 |

```
  [4] 学生请假审批
```

### 1. 查看我的授课日程（课表）

- 自动加载当前学期（2025-2026-1）该教师的全部排课记录
- 以表格形式展示：**星期 / 时间段 / 授课科目 / 授课班级 / 教学楼教室 / 校区**
- 示例：

```

星期一  第 1-2 节  Java程序设计     计算机2301班   A201       主校区
```

### 2. 学生成绩录入与管理

```
学生成绩控制中心
  [1] 录入 / 修改学生成绩（逐个录入）
  [2] 批量录入成绩（表格模式）
  [3] 查看成绩单与综合统计
  [4] 导出成绩单到 CSV
```

操作前需先选择本学期自己负责的教学计划（即哪个班的哪门课）。

| 功能 | 详细说明 |
|------|----------|
| **逐个录入** | 从课程学生列表选择一名学生，输入分数（0-100），系统自动换算等级（A/B/C/D/F） |
| **批量录入** | 按顺序逐一为班级所有学生输入分数，支持跳过（留空），完成后统一确认保存 |
| **成绩统计** | 展示全班成绩单，包含平均分、最高分、最低分、及格率、优秀率、分数段文本柱状图 |
| **导出 CSV** | 将该课程完整成绩单导出为 CSV 文件，方便线下存档 |

**成绩等级换算规则**：

| 分数段 | 等级 |
|--------|------|
| 90 ~ 100 | A |
| 80 ~ 89 | B |
| 70 ~ 79 | C |
| 60 ~ 69 | D |
| 0 ~ 59 | F |

## 四、学生端（student1 ~ student56）

学生只能查看和操作与自己相关的数据，主菜单共 6 项。

```
学生服务终端 — 当前学生: 李明（S230101）
  [1] 查看我的成绩单（与 GPA 计算）
  [2] 自主选课与退课（选修 / 公选）
  [3] 查看我的班级课表
  [4] 查看我的考勤统计
  [5] 提交请假申请与记录
  [6] 查看系统公告栏
```

---

### 1. 查看我的成绩单（与 GPA 计算）

- 展示当前学期所有已登分课程的成绩单
- 显示字段：课程编号 / 课程名称 / 学分 / 分数 / 成绩等第 / 考试类型
- 底部自动计算并展示 **学期 GPA**（满分 4.0，基于期末等第成绩）

**GPA 计算规则**：

| 等级 | 绩点 |
|------|------|
| A | 4.0 |
| B | 3.0 |
| C | 2.0 |
| D | 1.0 |
| F | 0.0 |

> GPA = Σ(课程学分 × 绩点) / Σ课程学分

---

### 2. 自主选课与退课

仅支持类型为 **选修课** 和 **公选课** 的课程，必修课由管理员统一安排。

```
自主选课系统
  [1] 查看我已选的课程
  [2] 查看可选的选修与公选课
  [3] 选修新课程
  [4] 退选已选课程
```

| 功能 | 详细说明 |
|------|----------|
| **查看已选课程** | 展示本学期已选全部课程：课程名/类型/学分/主讲教师/来源（自主选修 or 班级课程），共选 N 门汇总 |
| **查看可选课** | 列出当前学期全部可选的选修/公选课程：课程名/类型/学分/主讲老师/容量/已选人数 |
| **选课** | 从可选课列表选择后完成选课，系统验证：是否已选、是否已满员、**与已有课表（含必修课和其他已选选修课）是否时间冲突** |
| **退课** | 仅支持退选自主选修的选修/公选课，班级必修课无法退选，确认后退课 |

---

### 3. 查看我的班级课表

- 自动加载当前学期本人所在班级的完整课表
- 展示：**星期 / 时间段 / 课程名称 / 任课教师 / 教室 / 校区**

---

### 4. 查看我的考勤统计

- 汇总本学期各课程的考勤记录
- 展示字段：课程名称 / 出勤次数 / 迟到次数 / 早退次数 / 旷课次数 / 请假次数 / **出勤率**
- 可查看具体某门课的详细考勤记录（日期 + 状态 + 备注）

---

### 5. 提交请假申请与记录

```
请假管理
  [1] 提交新的请假申请
  [2] 查看我的请假记录
```

| 功能 | 详细说明 |
|------|----------|
| **提交申请** | 填写请假起止日期（yyyy-MM-dd 格式）和请假原因，确认后提交，状态默认为"待审批" |
| **查看记录** | 展示历史请假申请：时间段、原因、当前状态（待审批/已批准/已拒绝）、审批人和备注 |

---

### 6. 查看系统公告栏

- 展示所有面向学生或全员的公告通知
- 置顶公告优先显示，按发布时间倒序排列
- 显示：标题 / 发布人 / 发布时间 / 正文内容

## 五、关键交互特性

### 输入容错

所有输入均有校验，非法输入不会崩溃，提示重新输入：

```
请输入分数(0-100) > 150
[错误] 分数必须在 0-100 之间，请重新输入 >

确认删除该学生？(y/n) > x
[错误] 请输入 y 或 n >
```

### 删除确认

所有不可逆操作（删除、批量修改）均显示操作摘要并要求 `y/n` 二次确认。

### 表格输出

数据以等宽表格形式展示，中文字符按双倍宽度计算，保证对齐：

```
---- 计算机2301班 学生列表（共10人）----
序号 | 学号      | 姓名     | 性别 | 手机号
-----+-----------+----------+------+-------------
  1  | S230101   | 李明     | 男   | 138xxxxxxxx
  2  | S230102   | 张强     | 男   | 139xxxxxxxx
```

### 成绩柱状图

成绩分析以文本柱状图直观展示：

```
---- 高等数学 成绩分布 ----
  90-100: ████████████ 12人 (26.7%)
  80-89 : ██████████████████ 18人 (40.0%)
  70-79 : ██████████ 10人 (22.2%)
  60-69 : ████ 4人 (8.9%)
   0-59 : █ 1人 (2.2%)
---------------------------------
  平均分: 78.6  最高: 98  最低: 45  及格率: 97.8%
```
